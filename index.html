About This Platform

This is a demonstration of a P2P Trading Platform built with React Native, Tailwind CSS, and Node.js.

The platform features an escrow system for secure trading, real-time chat, multi-currency wallet, and dispute resolution system.

For a complete experience, you would need to run the React Native app on a mobile device or emulator.

The server API is running at /api
